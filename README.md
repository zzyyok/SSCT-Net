# SSCT-Net
Spatial–Spectral Collaborative CNN–Transformer Network (SSCT-Net) for Hyperspectral Image Super-Resolution.

## Features
- Dual-branch CNN–Transformer hybrid architecture
- SpectralSparseTransformer for spectral attention
- Swin-like Transformer for spatial enhancement
- MRCB multi-resolution reconstruction block
- Supports GPU acceleration (CUDA)
- Compatible with CAVE hyperspectral dataset

## Dataset
This repo supports the [CAVE hyperspectral dataset](https://www.cs.columbia.edu/CAVE/databases/multispectral/).
